# Rocket Elevators Operations Dashboard

This project develops a static dashboard prototype for analyzing Rocket Elevators operations data. Through data exploration and LLM-assisted analysis, the project demonstrates how AI can provide insights into elevator maintenance patterns, operational efficiency, and service trends.

## Repository Structure

- **platform/**: HTML and frontend code for the dashboard prototype
- **intelligence/**: Jupyter notebooks and Python scripts for data exploration and LLM prompting
- **data/**: Raw Rocket Elevators datasets used in the analysis
- **docs/**: Project specifications and reports, including the AI interaction log
